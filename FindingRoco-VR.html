VR experience

Finding Roco

This is an academic project I did with 4 other team members. We designed a VR game featuring 3 levels and puzzle solving, aiming to explore different types of interaction in the immersive environment. We developed this game on Oculus Rift using Unity and created essential 3D assets using Maya.

During the development process, we also did some usability testings to see how they feel and react within the VR environment. Based on the feedback, we were able to modify our design like movement speed, the scale of items.

Role: Design VR experience and interaction Unity Programming

Client: Academic project

Date: 2018 Spring

Skills: Interaction Design, Game Design, Programming (C#)
